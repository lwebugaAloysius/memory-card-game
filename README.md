## 🎮 About The Game

A simple **memory card matching game** built while learning React.

Flip cards, remember positions, and match all pairs to win.

---

## ✨ Features

* 🍓 Flip cards to reveal fruits
* 🎯 Match identical pairs
* 🔢 Move counter
* 🏆 Score tracking
* 🎉 Win detection
* 🔄 Restart game anytime
* 🎊 Confetti celebration on win

---

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/lwebugaAloysius/memory-card-game.git
```

### 2. Go into project

```bash
cd memory-card-game
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run development server

```bash
npm run dev
```
